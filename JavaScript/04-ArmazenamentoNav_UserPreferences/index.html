User Preferences

Sistema de Preferências de Usuário no Navegador

🎉 Olá, ! Boas-vindas!

Este é o projeto UserPreferences, desenvolvido para demonstrar na prática como funciona o armazenamento nativo no navegador utilizando Local Storage, Session Storage e Cookies.

O que você pode fazer:

  • 🎨 Personalizar o tema do site (claro ou escuro)
  • 🔐 Fazer um login simples que permanece enquanto esta aba estiver aberta
  • 🔔 Escolher se deseja visualizar notificações nas próximas visitas

Este projeto faz parte do meu estudo em JavaScript, com foco na criação de aplicações que oferecem uma melhor experiência ao usuário através do armazenamento local.
