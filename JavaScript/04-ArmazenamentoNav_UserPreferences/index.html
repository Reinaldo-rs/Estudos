UserPreferences

Sistema de Preferências de Usuário no Navegador

🎉 Olá, João! Boas-vindas! Este é o projeto UserPreferences Pro, desenvolvido para demonstrar na prática como funciona o armazenamento nativo no navegador utilizando Local Storage, Session Storage e Cookies. Aqui você pode: Personalizar o tema do site (claro ou escuro). Fazer um login simples que permanece enquanto esta aba estiver aberta. Escolher se deseja ou não visualizar nossa mensagem de boas-vindas nas próximas visitas.


Este projeto faz parte do meu estudo em JavaScript, com foco na criação de aplicações que oferecem uma melhor experiência ao usuário através do armazenamento local.
